скрипты, стили, картинки итд добавлять файлами в проект и тогда подключать (сдн не использовать)

все доп элементы (ссылки, картинки...) добавлять внутри "ли" элемента внутри списка

у ul, ol, li есть свои встроенные отступы

при создании многоуровнего списка- новый список вкладывать в li элемент

в css избегать большого уровня вложенности элементов (индивидуальные классы итд) =>упрощает читабельность кода

виды списков В ФОРМЕ: option, select

для изображения всегда указывать альт

не использовать float
